搭建Trojan科学上网节点

2020-03-30
科学上网

  • trojan是什么?

    trojan是一款科学上网工具

  • trojan的工作原理是什么?

    trojan服务端接收请求, 一旦发现请求来自浏览器, 就会将请求转发到Nginx, 模拟HTTPS情况下的web服务器. 如果发现进来的请求来自trojan客户端, 那就正常进行科学上网的操作.

  • 如何配置trojan?

    下载trojan编译好的包. 下载地址: 点我下载

  • 准备一个http服务器, Ubuntu下, 直接: sudo apt install nginx 就阔以啦😁

  • 搞个域名, 将域名解析到自己的VPS上.

  • 搞个证书, 最好嘛, 申请一个真正的证书, 装就装到底呗~ 😎
